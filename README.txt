IN THE BOOTH - FINAL PRODUCT PACKAGE

Project Type:
Website UI Prototype for McCall Southern Photography

Purpose:
This package contains the final product files for the In The Booth website prototype. The site demonstrates a customer-facing photography website with sample pages and UI workflows for booking, ordering, payment, touch-up requests, cropping, customer chat, and confirmations.

Technical Notes:
- Static HTML/CSS/JavaScript prototype
- Forms and payments are sample interface only
- Buttons navigate between pages where appropriate
- No back-end processing is included in this product package
